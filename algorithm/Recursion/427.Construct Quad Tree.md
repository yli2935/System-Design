# 427. Construct Quad Tree（建立四叉树）

> LeetCode 427 | 难度：Medium  
> 时间复杂度：最坏 **O(n² log n)**（每次 `allSame` 扫描当前子矩阵；若改用二维前缀和判同质可降至 O(n²)）| 空间：递归栈 O(log n)，树 O(叶子数)

---

## 一、问题描述

给定一个 `n × n` 的网格 `grid`，`n` 为 2 的幂，每个格子值为 `0` 或 `1`。要求用该网格构建一棵 **四叉树**（Quad-Tree），并返回根节点。

### 四叉树结点语义（本题定义）

| 属性 | 含义 |
|------|------|
| `isLeaf` | 当前结点表示的子区域 **全部为 0 或全部为 1** 时为 `true` |
| `val` | 叶子时表示该区域的统一值；非叶子时题目允许任意（常用 `true`） |
| `topLeft, topRight, bottomLeft, bottomRight` | 将当前正方形区域 **四等分** 后的四个子结点；叶子时均为 `null` |

```
若 2×2 全为 1 → 一个叶子 isLeaf=true, val=true
若 2×2 为 [[1,1],[0,0]] → 根 isLeaf=false，四个子结点各为叶子
```

---

## 二、问题本质：分治 + 递归

- 把 `[row, col]` 为左上角、边长为 `size` 的正方形看成一棵子树的「管辖区域」。
- **终止条件**：该区域内所有格子相同 → 压缩成 **一个叶子**。
- **否则**：边长折半，对左上、右上、左下、右下 **四个象限** 递归建树，当前结点为 **内部结点**。

### 子区域「是否全相同」

- **暴力**：对每次递归扫描 `size × size`，最坏层数约 `log n`，总代价约 **O(n² log n)**。
- **优化**：预处理 **二维前缀和**，则任意矩形内 1 的个数 **O(1)** 查询；若个数为 0 或面积为 `size*size` 则全相同，整体 **O(n²)**。

---

## 三、递归定义（口胡版）

```
build(row, col, size):
  if 区域 [row,col] 边长 size 内全 0 或全 1:
      return 叶子(val = 0 或 1)
  half = size / 2
  return 内部结点，四孩子分别为：
      build(row, col, half)              // 左上
      build(row, col + half, half)       // 右上
      build(row + half, col, half)       // 左下
      build(row + half, col + half, half) // 右下
```

**边界**：`size == 1` 时必为叶子。

## 四、完整代码

### Java（`helper` + `allSame` 左闭右开区间）

子矩阵用 **左闭右开** `[xStart, xEnd) × [yStart, yEnd)` 表示，与 `i < xEnd`、`j < yEnd` 的二重循环一致；`midX`、`midY` 将当前正方形四等分后递归。

```java
/*
// Definition for a QuadTree node.
class Node {
    public boolean val;
    public boolean isLeaf;
    public Node topLeft;
    public Node topRight;
    public Node bottomLeft;
    public Node bottomRight;

    public Node() {
        this.val = false;
        this.isLeaf = false;
        this.topLeft = null;
        this.topRight = null;
        this.bottomLeft = null;
        this.bottomRight = null;
    }

    public Node(boolean val, boolean isLeaf) {
        this.val = val;
        this.isLeaf = isLeaf;
        this.topLeft = null;
        this.topRight = null;
        this.bottomLeft = null;
        this.bottomRight = null;
    }

    public Node(boolean val, boolean isLeaf, Node topLeft, Node topRight, Node bottomLeft, Node bottomRight) {
        this.val = val;
        this.isLeaf = isLeaf;
        this.topLeft = topLeft;
        this.topRight = topRight;
        this.bottomLeft = bottomLeft;
        this.bottomRight = bottomRight;
    }
}
*/

class Solution {
    public Node construct(int[][] grid) {
        return helper(grid, 0, 0, grid.length, grid.length);
    }

    private Node helper(int[][] grid, int xStart, int yStart, int xEnd, int yEnd) {
        if (allSame(grid, xStart, yStart, xEnd, yEnd)) {
            return new Node(grid[xStart][yStart] == 1 ? true : false, true);
        }
        int midX = (xStart + xEnd) / 2;
        int midY = (yStart + yEnd) / 2;

        Node node = new Node(true, false);
        node.topLeft = helper(grid, xStart, yStart, midX, midY);
        node.topRight = helper(grid, xStart, midY, midX, yEnd);
        node.bottomLeft = helper(grid, midX, yStart, xEnd, midY);
        node.bottomRight = helper(grid, midX, midY, xEnd, yEnd);
        return node;
    }

    public Boolean allSame(int[][] grid, int xStart, int yStart, int xEnd, int yEnd) {
        int temp = grid[xStart][yStart];
        for (int i = xStart; i < xEnd; i++) {
            for (int j = yStart; j < yEnd; j++) {
                if (grid[i][j] != temp) {
                    return false;
                }
            }
        }
        return true;
    }
}
```

### JavaScript（暴力判断全相同，写法短）

```javascript
/**
 * // Definition for a QuadTree node
 * function Node(val,isLeaf,topLeft,topRight,bottomLeft,bottomRight) {
 *    this.val=val;
 *    this.isLeaf=isLeaf;
 *    this.topLeft=topLeft;
 *    this.topRight=topRight;
 *    this.bottomLeft=bottomLeft;
 *    this.bottomRight=bottomRight;
 * };
 */

/**
 * @param {number[][]} grid
 * @return {Node}
 */
var construct = function (grid) {
    const n = grid.length;

    function same(r, c, len) {
        const v = grid[r][c];
        for (let i = r; i < r + len; i++) {
            for (let j = c; j < c + len; j++) {
                if (grid[i][j] !== v) return null;
            }
        }
        return v === 1;
    }

    function build(r, c, len) {
        const uniform = same(r, c, len);
        if (uniform !== null) {
            return new Node(uniform, true, null, null, null, null);
        }
        const half = len >> 1;
        return new Node(
            true,
            false,
            build(r, c, half),
            build(r, c + half, half),
            build(r + half, c, half),
            build(r + half, c + half, half)
        );
    }

    return build(0, 0, n);
};
```

## 五、小结

| 要点 | 说明 |
|------|------|
| 为何算递归 | 同一规则作用在越来越小的子方阵上，天然 **自相似** |
| 剪枝 | 子区域同质则不再四分，对应四叉树 **压缩** |
| 与地理索引 | 二维空间四分与 **Quadtree** 一致，常用来做区域聚合与快速查询（本题只考建树） |
