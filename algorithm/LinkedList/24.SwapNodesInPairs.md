# 24. Swap Nodes in Pairs（两两交换链表中的节点）

> LeetCode 24 | 难度：Medium  
> 时间复杂度：O(n) | 空间复杂度：O(1)

---

## 一、问题描述

给定一个链表，**两两交换**其中相邻的节点，并返回交换后的链表头节点。

**要求**：不能只是修改节点内部的值，而是需要实际进行节点交换。

```
输入：head = [1, 2, 3, 4]
输出：[2, 1, 4, 3]

输入：head = []
输出：[]

输入：head = [1]
输出：[1]
```

---

## 二、核心思想：哑节点 + 三指针重连

### 为什么要 dummy？

头节点会在交换后变成第二个节点，用 `dummy.next` 始终指向真正的头，避免特判。

### 循环不变量

- `prev`：已处理段末尾，其 `next` 指向 **当前要交换的一对里原来的第一个**（`current`）
- `current`：当前对的第一个节点
- `current.next`（记为 `next`）：当前对的第二个节点

### 交换四步（重连指针）

对一对节点 `(current, next)`：

1. `current.next = next.next` — 第一个接到「下一对」的第一个
2. `prev.next = next` — 上一段接到新的头（原来的第二个）
3. `next.next = current` — 第二个指回原来的第一个
4. 移动：`prev = current`，`current = current.next`（即下一对的第一个）

```
交换前:  ... → prev → current → next → rest → ...
交换后:  ... → prev → next → current → rest → ...
```

## 三、完整代码

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode swapPairs(ListNode head) {
        ListNode dummy = new ListNode(0);
        dummy.next = head;

        ListNode prev = dummy;
        ListNode current = head;

        while (current != null && current.next != null) {

            ListNode next = current.next;

            current.next = next.next;
            prev.next = next;
            next.next = current;

            prev = current;
            current = current.next;
        }
        return dummy.next;
    }
}
```

---

## 四、执行过程示例

`head = 1 → 2 → 3 → 4 → null`

| 步骤 | prev | current | 链表（从 dummy 看） |
|------|------|---------|-------------------|
| 初始 | dummy | 1 | 0→1→2→3→4 |
| 交换 1,2 后 | 1 | 3 | 0→2→1→3→4 |
| 交换 3,4 后 | 3 | null | 0→2→1→4→3 |

返回 `dummy.next`：**2 → 1 → 4 → 3**

---

## 五、边界情况

| 情况 | 行为 |
|------|------|
| `head == null` | `while` 不进入，返回 `dummy.next` 即 `null` |
| 只有一个节点 | `current.next == null`，不交换，返回原头 |
| 奇数个节点 | 最后一对只有一个节点，循环结束，该节点保持原位 |

---

## 六、总结

| 要点 | 内容 |
|------|------|
| **技巧** | 哑节点统一头指针 |
| **循环条件** | `current != null && current.next != null` |
| **指针顺序** | 先保存 `next`，再改 `current.next`，再改 `prev.next` 与 `next.next`，避免丢链 |
| **推进** | `prev = current`，`current = current.next` |
