# 918. Maximum Sum Circular Subarray（环形子数组的最大和）

> LeetCode 918 | 难度：Medium  
> 时间复杂度：O(n) | 空间复杂度：O(1)

---

## 一、问题描述

给定一个 **环形** 整数数组 `nums`，找出数组中非空子数组的最大和。

环形数组意味着数组的末端连接到开头，即 `nums[n-1]` 的下一个元素是 `nums[0]`。

```
输入：nums = [1, -2, 3, -2]
输出：3
解释：子数组 [3] 有最大和 3

输入：nums = [5, -3, 5]
输出：10
解释：子数组 [5, 5]（环绕）有最大和 5 + 5 = 10

输入：nums = [-3, -2, -3]
输出：-2
解释：子数组 [-2] 有最大和 -2
```

---

## 二、核心思想：两种情况

环形数组的最大子数组和只有 **两种情况**：

### 情况一：最大子数组在中间（不跨越边界）

```
[  ...  |  max_subarray  |  ...  ]
        ←───────────────→
```

这就是普通的 **LeetCode 53**，用 Kadane 算法求解。

### 情况二：最大子数组跨越边界（两端相连）

```
[  max_part  |  min_subarray  |  max_part  ]
←───────────→                 ←───────────→
     左端                           右端
```

**关键洞察**：

如果最大子数组 **跨越边界**，那么 **中间剩下的部分** 就是 **最小子数组**！

```
max_circular = total_sum - min_subarray
```

---

## 三、算法公式

```
答案 = max(情况一, 情况二)
     = max(max_subarray, total_sum - min_subarray)
```

### 图解

```
情况一（不跨界）：
┌─────────────────────────────┐
│    [  max  ]                │
└─────────────────────────────┘

情况二（跨界）：
┌─────────────────────────────┐
│ max ]      [  min  ]    [max│
└─────────────────────────────┘
  ↑                         ↑
  └─────── 这两部分相连 ─────┘
  
  max_circular = total - min
```

---

## 四、边界情况：全是负数

如果数组全是负数：
- `max_subarray` = 最大的负数（比如 -2）
- `min_subarray` = 整个数组的和
- `total_sum - min_subarray` = 0（空数组）

但题目要求 **非空子数组**，所以不能选空数组。

**处理方式**：当 `max_subarray < 0` 时，直接返回 `max_subarray`。

---

## 五、变量含义

| 变量 | 含义 |
|------|------|
| `maxSum` | 当前位置结尾的最大子数组和 |
| `minSum` | 当前位置结尾的最小子数组和 |
| `maxTotal` | 全局最大子数组和（情况一答案） |
| `minTotal` | 全局最小子数组和 |
| `total` | 数组总和 |

---

## 六、完整代码

```javascript
function maxSubarraySumCircular(nums) {
    let total = 0;
    
    // Kadane for max subarray
    let maxSum = nums[0];
    let maxTotal = nums[0];
    
    // Kadane for min subarray
    let minSum = nums[0];
    let minTotal = nums[0];
    
    // 从第一个元素开始初始化 total
    total = nums[0];
    
    for (let i = 1; i < nums.length; i++) {
        const n = nums[i];
        total += n;
        
        // 最大子数组（Kadane）
        maxSum = Math.max(n, maxSum + n);
        maxTotal = Math.max(maxTotal, maxSum);
        
        // 最小子数组（Kadane 变体）
        minSum = Math.min(n, minSum + n);
        minTotal = Math.min(minTotal, minSum);
    }
    
    // 边界情况：全是负数
    if (maxTotal < 0) {
        return maxTotal;
    }
    
    // 返回两种情况的最大值
    return Math.max(maxTotal, total - minTotal);
}
```

### 更简洁的写法

```javascript
function maxSubarraySumCircular(nums) {
    let total = 0;
    let maxSum = -Infinity, minSum = Infinity;
    let curMax = 0, curMin = 0;
    
    for (const n of nums) {
        total += n;
        
        curMax = Math.max(curMax + n, n);
        maxSum = Math.max(maxSum, curMax);
        
        curMin = Math.min(curMin + n, n);
        minSum = Math.min(minSum, curMin);
    }
    
    // 全负数情况：maxSum < 0，此时 total === minSum
    return maxSum > 0 ? Math.max(maxSum, total - minSum) : maxSum;
}
```

## 七、执行过程示例

### 示例 1：`nums = [5, -3, 5]`

| i | n | curMax | maxSum | curMin | minSum | total |
|---|---|--------|--------|--------|--------|-------|
| 0 | 5 | 5 | 5 | 5 | 5 | 5 |
| 1 | -3 | 2 | 5 | -3 | -3 | 2 |
| 2 | 5 | 7 | 7 | 2 | -3 | 7 |

- 情况一：`maxSum = 7`（子数组 [5, -3, 5]）
- 情况二：`total - minSum = 7 - (-3) = 10`（子数组 [5, 5] 跨界）

最终答案：`max(7, 10) = 10` ✓

---

### 示例 2：`nums = [1, -2, 3, -2]`

| i | n | curMax | maxSum | curMin | minSum | total |
|---|---|--------|--------|--------|--------|-------|
| 0 | 1 | 1 | 1 | 1 | 1 | 1 |
| 1 | -2 | -1 | 1 | -2 | -2 | -1 |
| 2 | 3 | 3 | 3 | 1 | -2 | 2 |
| 3 | -2 | 1 | 3 | -1 | -2 | 0 |

- 情况一：`maxSum = 3`
- 情况二：`total - minSum = 0 - (-2) = 2`

最终答案：`max(3, 2) = 3` ✓

---

### 示例 3：`nums = [-3, -2, -3]`（全负数）

| i | n | curMax | maxSum | curMin | minSum | total |
|---|---|--------|--------|--------|--------|-------|
| 0 | -3 | -3 | -3 | -3 | -3 | -3 |
| 1 | -2 | -2 | -2 | -5 | -5 | -5 |
| 2 | -3 | -3 | -2 | -8 | -8 | -8 |

- `maxSum = -2`（最大的负数）
- `total - minSum = -8 - (-8) = 0`（空数组，不合法）

因为 `maxSum < 0`，直接返回 `maxSum = -2` ✓

## 八、为什么这个思路正确？

### 数学证明

设环形数组为 `A[0], A[1], ..., A[n-1]`

**跨界子数组** 可以表示为：`A[i], A[i+1], ..., A[n-1], A[0], ..., A[j]`

其和为：
```
sum(A[i:n]) + sum(A[0:j+1])
= total - sum(A[j+1:i])
= total - (中间不选的部分)
```

要最大化跨界子数组和，就要 **最小化中间不选的部分**，即找 **最小子数组**。

## 十、总结

1. **核心洞察**：环形最大和 = max(不跨界最大, 跨界最大)
2. **跨界最大** = `total_sum - min_subarray`
3. **一次遍历**：同时用 Kadane 求最大子数组和最小子数组
4. **边界处理**：全负数时，`total - minSum = 0` 是空数组，需特判
5. **时间 O(n)，空间 O(1)**：只需要常数个变量

### 核心公式

```
答案 = max(maxSubarray, totalSum - minSubarray)

特判：if maxSubarray < 0, return maxSubarray
```
